Common HVAC Stock Parts Reference - For Inventory Assistant

Category: Capacitors
- Dual Run Capacitor, 35/5 MFD, 440V (Part# CAP-DR-3505-440) - Common for 2-3 Ton units
- Dual Run Capacitor, 40/5 MFD, 440V (Part# CAP-DR-4005-440) - Common for 3-4 Ton units
- Dual Run Capacitor, 45/5 MFD, 440V (Part# CAP-DR-4505-440) - Common for 4-5 Ton units
- Dual Run Capacitor, 50/7.5 MFD, 440V (Part# CAP-DR-5075-440) - Larger residential units
- Start Capacitor, 189-227 MFD, 330V (Part# CAP-ST-189-330) - Common hard start kit component
- Run Capacitor (Fan Motor), 5 MFD, 370/440V (Part# CAP-RN-05-440)
- Run Capacitor (Fan Motor), 7.5 MFD, 370/440V (Part# CAP-RN-07-440)
Notes: Always replace with exact MFD rating. Voltage rating can be equal or higher. Check physical dimensions for fit.

Category: Contactors
- Contactor, 1 Pole, 30 Amp (Part# CON-1P-30A) - Common for smaller condensers/heat pumps
- Contactor, 2 Pole, 30 Amp (Part# CON-2P-30A) - Widely used in residential AC units
- Contactor, 2 Pole, 40 Amp (Part# CON-2P-40A) - Used in larger residential units (4-5 Ton)
Notes: Check coil voltage (usually 24V). Inspect for pitted or welded contacts.

Category: Motors
- Condenser Fan Motor, 1/4 HP, 208/230V, 825 RPM (Part# MOT-CF-14HP-825) - Verify rotation (CW/CCW) and mounting.
- Condenser Fan Motor, 1/3 HP, 208/230V, 1075 RPM (Part# MOT-CF-13HP-1075)
- Blower Motor (Indoor), 1/2 HP, 115V, 4-Speed (Part# MOT-BL-12HP-115-4S) - PSC type, check rotation/frame size.
- Blower Motor (Indoor), 3/4 HP, Variable Speed ECM (Part# MOT-BL-ECM-34HP) - Model specific, requires careful matching.
Notes: Always match HP, Voltage, RPM, Frame Size, Rotation. Capacitor needed for PSC motors usually specified on motor label.

Category: Filters
- Standard Air Filter, 16x20x1 MERV 8 (Part# FIL-16201-M8)
- Standard Air Filter, 16x25x1 MERV 8 (Part# FIL-16251-M8)
- Standard Air Filter, 20x20x1 MERV 8 (Part# FIL-20201-M8)
- Standard Air Filter, 20x25x1 MERV 8 (Part# FIL-20251-M8)
- Media Filter, 16x25x4 MERV 11 (Part# FIL-16254-M11)
- Media Filter, 20x25x4 MERV 11 (Part# FIL-20254-M11)
Notes: Stock common sizes. Advise customers on correct size and replacement frequency.

Category: Refrigerant (Requires EPA Certification)
- R410a Refrigerant, 25lb Cylinder (Part# REF-R410A-25)
- R22 Refrigerant, 30lb Cylinder (Part# REF-R22-30) - Use restricted, for existing systems only.
Notes: Handle according to EPA regulations. Always use correct refrigerant type for the system.

Category: Thermostats
- Non-Programmable Digital Thermostat (Basic) (Part# TSTAT-DIG-NP)
- Programmable Thermostat (5-2 Day) (Part# TSTAT-DIG-P52)
- Wi-Fi Smart Thermostat (Part# TSTAT-WIFI-SM)
Notes: Check system compatibility (e.g., heat pump, conventional, stages).
